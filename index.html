O que é HTML?

(Voltar)
    • HTML é um arquivo de marcação, não é uma linguagem de programação.
    • São textos delmitados por nomes que o navegador consegue interpretar
    • Cada nome desses demilitadores, tem um nome e um comportamento específico.

    O que são Tags?

    As tags também são responsáveis ​​por criar elementos nas páginas da web com a função de organizar, identificar e limitar o conteúdo de uma página.

    Atributos HTML

    Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc

    Como Executar um Arquivo HTML

      1. Navegue até o arquivo HTML que você deseja abrir. Navegadores como o Chrome, Safari e Microsoft Edge exibem o código HTML como websites, e não no formato de edição. Use este método para visualizar o arquivo como página web.
      2. Clique com o botão direito no arquivo de extensão .htm ou .html. Em seguida, um menu de opções será expandido.
      3. Selecione o menu Abrir com. Em seguida, uma lista de programas instalados no computador será exibida.
      4. Selecione um navegador de internet. Algumas opções incluem Edge, Safari, Chrome e Firefox. Após selecionar um navegador de internet, ele será aberto, exibindo a página web

    Formatação de Textos com HTML

    (Voltar)

    Títulos

    As tags padrão para títulos que já estão pré-formatadas nos navegadores Web são: h1, h2, h3, h4, h5, h6 . Estas tags já estão formatadas em negrito e com uma letra maior do que a do texto convencional. A única diferença entre estas tags é que a h1 possui a maior tamanho de letra e a letra vai diminuindo de tamanho até chegar até a h6, que possui o menor tamanho de letra.

    Parágrafos

    A tag que tem a função de formatar um bloco de texto como parágrafo é a tag p. Ela é utilizada da mesma forma que as tags dos títulos.

    Elemento de Abreviatura

    O elemento ABBR "Abbreviation Element" ou abreviatura é o marcador utilizado para se inserir uma sigla em uma página HTML, ele pode acontecer, por exemplo, dentro de um parágrafo.

    É útil para fornecer informações aos navegadores, leitores de telas, buscadores, sites de tradução como também verificadores de ortografia.

    Exemplo de uma sigla inserida com o marcador ABBR

    A autoridade monetária do Brasil, abbr title="Banco Central do Brasil">BCB /abbr>

    A informação title="Banco Central do Brasil", não é visualizada no navegador, mas a informação do que representa BCB esta disponível para as ferramentas mencionadas acima.

    Outras Tags para Formatação de Textos

    Estilos de texto

    • Strong = Colocam o texto em negrito
    • i = Coloca o texto em itálico
    • u = Coloca um subinhado no texto
    • Sub = Coloca o texto como subscrito
    • Sup = Coloca o texto como Sobrescrito
    • Del = Coloca o texto rasurado

    Lista Ordenada e Não-Ordenada

    (Voltar)

    A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.

    • Lista Ordenada

      • Uma lista ordenada é delimitada pelas etiquetas OL e /OL (Ordered List). Cada item da lista é delimitado pelas etiquetas LI e /LI (List Item).

    • Lista não-ordenada

      • A estrutura da lista não-ordenada é semelhante a da lista ordenada - porém, ao invés de ser construída com o elemento OL, é construída com o elemento UL (Unordered List).

    Links

    (Voltar)

    O elemento link é utilizado para vincular conteúdos de outros arquivos na página de código HTML. Ele deve ser inserido entre as tags head… da página e é bastante utilizado para adicionar arquivos de folha de estilos CSS. Vale ressaltar que não é necessário utilizar a tag de fechamento como acontece com outros elementos HTML.
